# NewsPulse 📰

NewsPulse is a responsive news web application built using HTML, CSS, Bootstrap, and Vanilla JavaScript.  
It fetches real-time news data from an external REST API and dynamically displays it in a clean card-based layout.

---

## 🚀 Features

- Category-based news filtering  
- Keyword-based search functionality  
- Dynamic UI rendering using JavaScript  
- Fallback image handling for missing news images  
- Fully responsive design using Bootstrap  

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- Bootstrap 5  
- Vanilla JavaScript  
- REST API (MediaStack / GNews API)  

---

## ⚙ How to Run Locally

1. Download or clone this repository  
2. Open `index.html` in your browser  
3. Add your API key inside `index.js`
